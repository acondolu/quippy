# quippy

_Quippy_ is an end-to-end encrypted expense splitter. Here is how to use the name in a sentence:
- "The restaurant was 10€. Let me quipp-it."
- "Let's quippy the summer holidays!"

Most notable features:
- Split expenses among any number of people
- No registration needed
- Progressive Web App: runs on desktop as well as mobile
- Supports dark mode 😎
- End-to-end encryption, meaning only the participants of an expense list can access its details.
